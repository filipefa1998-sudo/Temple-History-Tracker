Bulk Import from a Spreadsheet (CSV)

The fastest way to load your roster. Put names in a spreadsheet, save as CSV, and drop it here. A Name column is all you need to start; a Household column is recommended so families group together. These columns are recognized automatically (any order, extras ignored): Name, Household, Phone, Prep Status, Endowment Status, Submitting Names, Indexing, Generations, FH Group, Role, Notes.
